## Настройка

Убедимся, что лишние файлы не будут установлены:

```bash
sed -i 's/extras//' Makefile.in
```

Выполним скрипт `configure`:

```bash
./configure --prefix=/usr   \
            --host=$LIN_TGT \
            --build=$(./config.guess)  --disable-nls
```

## Сборка

```bash
make
```

## Установка

```bash
make DESTDIR=$LIN install
```
